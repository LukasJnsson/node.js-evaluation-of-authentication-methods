Scenario 1

Traditional username and password

Description

In this scenario, you will interact with a traditional login system, the most widely used method of authentication across various online platforms.

Task 1: Registration

Enter the information for all fields: first name, last name, a secret about you, email, and password. The password must be a minimum of 8 characters. The data does not have to be real, but you must remember the email and password to log in. Once all fields have the required input, you can register your account by clicking on the "register" button.

Task 2: Login

By using the email and password you created during registration, type the email in the email field and the password in the password field. Then click on the "login" button to log in.

Privacy Notice

The data will always be able to be read by the company. This is because there is no end-to-end encryption option for traditional email and password login in this scenario. The researchers remove all credentials after the experiment is done.
